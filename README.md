## preonboarding-assignment1-hayanmind

### 소개
반복적으로 Comment를 10개 씩 받아, 무한 스크롤로 Comment 목록을 구현

### 구현 목록
* Comment 목록 로딩
* 무한 스크롤로 Comment 10개씩 반복 로딩

### 설치 및 시작
```
npm install
npm start
```
